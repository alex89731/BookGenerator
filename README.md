# BookGenerator 

Автоматизированный генератор полноценных художественных книг с помощью нейросетей (Google Gemini) и типографской системы Typst.

## О проекте
BookGenerator — это десктопное приложение с графическим интерфейсом, созданное для генерации объемных, связных и логичных текстовых произведений. Программа берет вашу идею и "Библию проекта" (лор), после чего создает детальный план книги и поэтапно пишет её, сцену за сценой, сохраняя контекст и непрерывность повествования. Итоговый результат автоматически верстается и сохраняется в формате PDF.

## Основные возможности 
* **Умное планирование:** Агент-Архитектор создает строгую, непрерывную структуру сюжета с разбивкой на главы и сцены.
* **Жесткая связность (Континуитет):** Генерация происходит по принципу неразрывного потока времени — каждая новая сцена начинается ровно там, где закончилась предыдущая.
* **Долгосрочная память:** Агент-Хранитель Памяти ведет подробную летопись сюжета, обновляя выжимку после каждой главы, чтобы ИИ не забывал важные детали, инвентарь героев и их текущее местоположение.
* **Управление API Gemini:** 
  * Автоматическая обработка серверных ошибок (500-е коды) и лимитов запросов (429 Quota).
  * Умная система повторных попыток с настраиваемыми задержками.
  * Обход базовых блокировок безопасности для свободы творчества (автоматически выставлены настройки `BLOCK_NONE`).
* **Автоматическая верстка:** Программа сама генерирует код и компилирует готовый, красиво оформленный PDF. (Система сама скачает портативный компилятор Typst, если он не установлен).
* **Графический интерфейс:** Удобный UI на базе Tkinter для комфортного взаимодействия, отслеживания логов и настройки генерации.

## Используемые технологии 
* **Python 3** — Основная логика программы.
* **Tkinter** — Построение графического интерфейса пользователя (GUI).
* **Google Generative AI (Gemini API)** — LLM-движок для генерации текста.
* **Typst** — Современная профессиональная система верстки для генерации финального PDF (альтернатива LaTeX).

## Установка и запуск 

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/alex89731/BookGenerator.git
   cd BookGenerator
   ```
2. Создайте виртуальное окружение (рекомендуется) и установите зависимости:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Для Windows
   pip install google-generativeai
   ```
   *(Tkinter обычно поставляется в стандартной библиотеке Python)*
3. Запустите программу:
   ```bash
   python main.py
   ```

## Как использовать 
1. Получите API-ключ на [Google AI Studio](https://aistudio.google.com/).
2. Запустите приложение и вставьте ключ в соответствующее поле (или задайте переменную окружения `GEMINI_API_KEY`).
3. Нажмите "Загрузить модели" и выберите желаемую модель из выпадающего списка.
4. Заполните основные поля:
   * **Название книги**
   * **Библия проекта:** Лор, константы, правила мира, описание главных героев.
   * **Запрос:** Сюжет и идея конкретной книги.
5. Нажмите "Начать генерацию (Много сцен)". 
6. В логе внизу экрана вы сможете следить за процессом работы различных агентов (планировщика, писателя, суммаризатора).
7. После окончания процесса сборки, программа предложит диалоговое окно для сохранения готового PDF-файла!

## Важное примечание 
Написание целой книги — процесс не быстрый из-за задержек между API-запросами (необходимых во избежание блокировок за превышение лимитов) и большого количества генерируемого текста. После запуска просто оставьте программу работать в фоне и следите за логами.
